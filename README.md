# vwechatpay

本项目包装微信支付 [go sdk](https://github.com/wechatpay-apiv3/wechatpay-go) 的一些支付能力，降低使用门槛。

## 功能特点

- 简化微信支付API的调用流程
- 提供更友好的接口封装
- 支持普通商户和服务商模式
